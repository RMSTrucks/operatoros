# Maps of Content

Create MOCs as your knowledge grows. A MOC is a navigation hub for a topic area.

## How MOCs Work

1. Start writing notes in `notes/` about things you learn
2. When you have 5+ notes on a related topic, create a MOC
3. The MOC links to those notes and provides narrative structure
4. MEMORY.md links to MOCs, MOCs link to notes = progressive disclosure

## MOC Template

```markdown
---
title: [Topic] MOC
description: [One sentence covering what this MOC navigates]
type: moc
created: YYYY-MM-DD
---

# [Topic]

[1-2 paragraph intro -- what this area covers and why it matters]

## [Section 1]
- [[note-name]] -- brief context
- [[note-name]] -- brief context

## [Section 2]
- [[note-name]] -- brief context
```

## Example MOCs

- **Infrastructure** -- services, databases, networking, deployment
- **Business** -- customers, revenue, operations, strategy
- **Architecture** -- design decisions, patterns, tech stack
